# ORVIXA UX CONSTITUTION
*Universal AI Learning & Interview Copilot*

---

## 1. VISIBLE PIXEL PURITY

- **No Marketing Banners:** Welcome cards, promotional texts, and upgrade banners are strictly forbidden.
- **Direct Utility:** Every element on the side panel must support the immediate learning objective. If a component does not help the current task, it must be removed.
- **Surgical Layouts:** No heavy headers, large company branding logos, or empty layout cards. Title area consists of a clean icon, category indicator, and compact context name.

---

## 2. ADAPTIVE SIDE PANEL MECHANICS

- **Right Dock Default:** The panel sits on the right side of the screen at a default width of 35% (range 25%–45%).
- **Smooth Layout Shift:** The host website shifts cleanly without overlapping. Zero CLS (Cumulative Layout Shift) is enforced.
- **Instant Shortcuts:** `Ctrl+K` (or `Cmd+K`) toggles the dock open or closed. The transition must complete under 150ms.
- **Clean Escape Hatch:** Pressing `Esc` immediately collapses the side panel and aborts any active streaming AI requests.

---

## 3. SCREENREAD CONTROL

- **No Automatic Scanning:** Orvixa never scans screen content without user action.
- **Compact Trigger Icon:** A clean, screen-analysis trigger icon is placed in the bottom control bar to initiate context capture, avoiding large text buttons.
