# LEARNING EXPERIENCE ARCHITECTURE
*Subject-Specific Adaptive Learning Cards & Intent Engine*

---

## 1. SUBJECT CARD COMPOSITION MATRIX

Orvixa composes adaptive learning cards based on subject matter and learner intent:

### 1. Programming & Computer Science
- `ConceptCard`: Core algorithm or data structure explanation.
- `CodeDiffTraceCard`: Interactive line-by-line syntax diff or execution trace.
- `CommonMistakesCard`: List of edge-case bugs and pointer pitfalls.
- `InterviewCard`: Mock coding interview question & follow-up prompts.

### 2. Physics & Engineering
- `ConceptCard`: Physical law or theorem statement.
- `FormulaCard`: Mathematical formulation with variable definitions.
- `NumericalCard`: Stepwise numerical calculation guide.
- `CommonMistakesCard`: Unit conversion or vector component errors.

### 3. English & Language Learning
- `MeaningCard`: Definition and nuance breakdown.
- `PronunciationCard`: Phonetic transcription and syllable stress.
- `UsageCard`: Sentence context examples.
- `PracticeCard`: Interactive prompt for student construction.

### 4. Mock Interview & Roleplay Mode
- `QuestionCard`: Realistic technical or behavioral interview question.
- `IdealAnswerCard`: Structured STAR framework solution.
- `WeakAnswerCard`: Examples of common candidate pitfalls.
- `FeedbackCard`: Real-time candidate evaluation & improvement tips.
